BirdOS Wlogout Wiki:

NOTE:To load the config than you make, make it on ~/.config/wlogout

Layout:

Layout is for managing what each button is doing

Example(This code is NOT for copy/paste):

{
    "label" : "shutdown", #The label
    "action" : "shutdown now", #What command this button will do, for this is to shutdown
    "text" : "Shutdown", #The text in the button
    "keybind" : "s" #The keybind press the button without arrow keys and Enter
    
}

Style:

The style(using CSS)to style the buttons and colors(style.css the file is called)

Example(This code is NOT for copy/paste):

* {
background-image: none; #If will be a wallpaper
font-size: 20px; #The font Size
font-family: "JetBrainsMono Nerd Font Regular"; #What font is using 
}



window {
	background-color: rgba(17, 17, 17, 0.45)
}
